# ✅ Lösungen zu den Blank-Vorlagen

Komplett ausgefüllte, lauffähige Lösungen zu den Übungsdateien in `../` (eine Ebene höher). Gleiche Dateinamen → einfaches 1:1-Vergleichen.

## So nutzen
1. Erst selbst in `../<datei>.py` schreiben.
2. `python3 Loesungen/<datei>.py` ausführen und die Ausgabe vergleichen.
3. Bei Abweichung Zeile für Zeile gegen die Lösung prüfen.

Die 🟨-Kommentare zeigen, welche Teile du selbst schreiben musstest; 🟩 war vorgegeben.

| Datei | erwartete Ausgabe (Kurz) |
|---|---|
| 01_testen_assertions | `[[1, 2, 3]]` / `[]` / AssertionError |
| 02_testen_pytest | „Alle Tests grün." (oder `python3 -m pytest`) |
| 03_datei_lesen_schreiben | Apfel / Banane / Kirsche |
| 04_sys_argv | `Summe: 12.0` (Aufruf: `... 3 4 5`) |
| 05_getopt_log | Zählung nach level/user/action in Ausgabedatei |
| 06_argparse_log | wie 05, nur mit argparse |
| 07_exceptions | 4 Fälle + je „Berechnung abgeschlossen." |
| 08_linkedlist | A C D |
| 09_stack_queue | pop: C / dequeue: A |
| 10_bst | 5..35 sortiert, search, delete(20) |
| 11_doubly | vorwärts A B C / rückwärts C B A |
| 12_sortierverfahren | alle: `[3, 7, 12, 42, 55, 78]` |
| 13_trie | `['car','cart','cat']` / `['dog','dot','dove']` |
| 14_ternaerer_suchbaum | `Hi, Dog, Cat, World, Hello, Python` |
| 15_circular_list | A C D |

> 05/06 brauchen eine `log.txt` im aktuellen Ordner (Format `zeit;LEVEL;user;action`), z.B. aus `../../WS2526/` kopieren.
