# 🛠️ Übungsplatz

Dein **freier Bereich zum Üben** – hier kannst du beliebige Dateien anlegen (Skizzen, eigene Lösungen, Blank-Page-Versuche, Experimente).

Vorschläge:
- Beim **Blank-Page-Üben**: Code aus dem Kopf hier reinschreiben (z.B. `stack_versuch.py`), ausführen, dann mit `2_Zusammenfassungen/3_Alle_Codes_zum_Abtippen.md` vergleichen.
- Beim **Altklausur-Üben**: eine Aufgabe aus `3_Altklausuren/*/Aufgabenstellung.pdf` hier lösen, dann mit der `Loesung/` abgleichen.
- Eigene Notizen, Merkzettel, Mini-Tests – alles erlaubt.

*(Dieser Ordner ist absichtlich leer. Lege einfach neue `.py`- oder `.md`-Dateien darin an. Zum Sichern auf GitHub: `git add -A && git commit -m "…" && git push`.)*
